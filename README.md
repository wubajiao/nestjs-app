## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

### nest g

```bash
# 创建一个控制器和该控制器的单元测试文件
nest g co posts

# module
nest g module gir

# controller
nest g controller girl --no-spec

# service**
nest g service girl --no-spec
```
